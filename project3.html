Project 3

Access Control & Security Gates

نظام بوابة أمنية يتحقق من البطاقة ويتعامل مع الطوارئ باستخدام 8086 و 8255

← الرجوع للصفحة الرئيسية

1. شرح المشكلة وفهم فكرتها

في بوابات الشركات أو مواقف السيارات أو الجامعات، لا يكفي أن تفتح البوابة بمجرد وجود شخص أو سيارة. يجب أن يتأكد النظام من وجود المستخدم أولاً، ثم يتحقق من أن البطاقة صحيحة، وبعدها فقط يسمح بفتح البوابة.

إذا كانت البطاقة غير صحيحة، يجب منع الدخول وتشغيل لمبة حمراء أو جرس تنبيه. وإذا حدثت حالة طوارئ، مثل ضغط زر Emergency Stop، يجب إيقاف البوابة فوراً وعدم السماح بأي حركة حتى تنتهي حالة الطوارئ.

فكرة المشروع ببساطة

8255 يستقبل إشارات الحساسات والبطاقة من Port A، ثم يرسل أوامر إلى الموتور واللمبات والجرس من Port B، ويعرض حالة البوابة من Port C.

2. شرح الحل باستخدام 8255

يتم برمجة Port A كمدخل Input. هذا المنفذ يقرأ وجود شخص أو سيارة، ويفحص هل البطاقة صحيحة أو غير صحيحة، ويفحص زر الطوارئ.

يتم برمجة Port B كمخرج Output للتحكم في موتور فتح البوابة، ولمبة السماح الخضراء، ولمبة الرفض الحمراء، والجرس. أما Port C فيستخدم كمخرج لعرض حالة البوابة مثل Gate Open أو Gate Locked أو Emergency.

البرنامج يعطي أولوية لحالة الطوارئ. فإذا كان زر الطوارئ مضغوطاً، يتم إيقاف الموتور وتشغيل التحذير. وإذا لم توجد طوارئ، يفحص النظام وجود شخص أو سيارة، ثم يفحص البطاقة. إذا كانت صحيحة، يفتح البوابة لفترة قصيرة ثم يقفلها. وإذا كانت غير صحيحة، يشغل الرفض والجرس.

3. Interface Map

Port A = Input PA0 = Person / Vehicle Sensor
PA1 = Valid Card Signal
PA2 = Invalid Card Signal
PA3 = Emergency Stop
Port B = Output PB0 = Gate Motor Relay
PB1 = Green LED
PB2 = Red LED
PB3 = Buzzer
Port C = Output PC0 = Gate Open Indicator
PC1 = Gate Locked Indicator
PC2 = Emergency Indicator

Control Word المستخدم هو 90H. معناه أن Port A مدخل Input، و Port B و Port C مخارج Output، وكلها تعمل في Mode 0.

4. الكود الكامل للمشكلة

ORG 100H

; -------------------------------
; 8255 Port Addresses
; -------------------------------
PORTA   EQU 300H
PORTB   EQU 301H
PORTC   EQU 302H
CTRL    EQU 303H

; -------------------------------
; Main Program
; -------------------------------
START:
    MOV DX, CTRL
    MOV AL, 90H
    OUT DX, AL

    ; Gate locked at beginning
    MOV DX, PORTB
    MOV AL, 00H
    OUT DX, AL

    MOV DX, PORTC
    MOV AL, 02H
    OUT DX, AL

MAIN_LOOP:
    ; Read gate sensors and card signals
    MOV DX, PORTA
    IN  AL, DX

    ; Emergency stop has highest priority
    TEST AL, 08H
    JNZ EMERGENCY_STOP

    ; Check if person or vehicle exists
    TEST AL, 01H
    JZ MAIN_LOOP

    ; Check valid card
    TEST AL, 02H
    JNZ ACCESS_GRANTED

    ; Check invalid card
    TEST AL, 04H
    JNZ ACCESS_DENIED

    JMP MAIN_LOOP

; -------------------------------
; Access Granted
; -------------------------------
ACCESS_GRANTED:
    ; Open gate motor + green LED
    MOV DX, PORTB
    MOV AL, 03H
    OUT DX, AL

    ; Gate open indicator
    MOV DX, PORTC
    MOV AL, 01H
    OUT DX, AL

    CALL GATE_DELAY

    ; Lock gate again
    MOV DX, PORTB
    MOV AL, 00H
    OUT DX, AL

    MOV DX, PORTC
    MOV AL, 02H
    OUT DX, AL

    JMP MAIN_LOOP

; -------------------------------
; Access Denied
; -------------------------------
ACCESS_DENIED:
    ; Red LED + buzzer
    MOV DX, PORTB
    MOV AL, 0CH
    OUT DX, AL

    CALL SHORT_DELAY

    ; Keep red LED only
    MOV DX, PORTB
    MOV AL, 04H
    OUT DX, AL

    CALL SHORT_DELAY

    ; Turn all off and keep gate locked
    MOV DX, PORTB
    MOV AL, 00H
    OUT DX, AL

    MOV DX, PORTC
    MOV AL, 02H
    OUT DX, AL

    JMP MAIN_LOOP

; -------------------------------
; Emergency Stop
; -------------------------------
EMERGENCY_STOP:
    ; Stop motor, turn red LED and buzzer ON
    MOV DX, PORTB
    MOV AL, 0CH
    OUT DX, AL

    ; Emergency indicator
    MOV DX, PORTC
    MOV AL, 04H
    OUT DX, AL

EMERGENCY_LOOP:
    CALL SHORT_DELAY

    ; Read Port A again
    MOV DX, PORTA
    IN  AL, DX

    ; Stay here while emergency switch is pressed
    TEST AL, 08H
    JNZ EMERGENCY_LOOP

    ; Return to locked state after emergency ends
    MOV DX, PORTB
    MOV AL, 00H
    OUT DX, AL

    MOV DX, PORTC
    MOV AL, 02H
    OUT DX, AL

    JMP MAIN_LOOP

; -------------------------------
; Delay while gate is open
; -------------------------------
GATE_DELAY PROC
    PUSH CX

    MOV CX, 30

GD_LOOP:
    CALL SHORT_DELAY
    LOOP GD_LOOP

    POP CX
    RET
GATE_DELAY ENDP

; -------------------------------
; Short Delay
; -------------------------------
SHORT_DELAY PROC
    PUSH CX

    MOV CX, 0FFFFH

D1:
    LOOP D1

    POP CX
    RET
SHORT_DELAY ENDP

END START

5. شرح الكود سطر بسطر

السطر الشرح
ORG 100H يحدد أن البرنامج من نوع COM ويبدأ من العنوان 100H.
PORTA EQU 300H يعطي اسم PORTA للعنوان 300H، وهو المنفذ المستخدم لقراءة الحساسات والبطاقة.
PORTB EQU 301H يعطي اسم PORTB للعنوان 301H، وهو المنفذ الذي يتحكم بالموتور واللمبات والجرس.
PORTC EQU 302H يعطي اسم PORTC للعنوان 302H، وهو منفذ مؤشرات حالة البوابة.
CTRL EQU 303H يعطي اسم CTRL للعنوان 303H، وهو Control Register الخاص بإعداد 8255.
START: بداية البرنامج.
MOV DX, CTRL يضع عنوان Control Register داخل DX.
MOV AL, 90H يضع Control Word في AL. القيمة 90H تعني Port A مدخل و Port B و Port C مخارج.
OUT DX, AL يرسل Control Word إلى 8255 حتى يتم ضبط اتجاه المنافذ.
MOV DX, PORTB يحدد Port B لأننا نريد ضبط مخارج البوابة في البداية.
MOV AL, 00H يضع صفر في AL، أي إطفاء الموتور واللمبات والجرس.
OUT DX, AL يرسل 00H إلى Port B، وبذلك تكون البوابة غير مفتوحة في البداية.
MOV DX, PORTC يحدد Port C لعرض حالة البوابة.
MOV AL, 02H يضع 02H في AL، أي تشغيل PC1 الذي يمثل حالة Gate Locked.
OUT DX, AL يرسل 02H إلى Port C، فتظهر حالة أن البوابة مقفلة.
MAIN_LOOP: بداية حلقة المراقبة الرئيسية.
MOV DX, PORTA يضع عنوان Port A في DX لقراءة الحساسات.
IN AL, DX يقرأ قيمة Port A ويضعها في AL. هذه القيمة تحتوي حالة الشخص والبطاقة والطوارئ.
TEST AL, 08H يفحص PA3. القيمة 08H تعني البت الرابع، وهو زر Emergency Stop.
JNZ EMERGENCY_STOP إذا كان زر الطوارئ مضغوطاً، ينتقل مباشرة إلى حالة الطوارئ.
TEST AL, 01H يفحص PA0 لمعرفة هل يوجد شخص أو سيارة عند البوابة.
JZ MAIN_LOOP إذا لا يوجد شخص أو سيارة، يرجع البرنامج للمراقبة ولا يفعل شيئاً.
TEST AL, 02H يفحص PA1، وهو إشارة البطاقة الصحيحة.
JNZ ACCESS_GRANTED إذا كانت البطاقة صحيحة، ينتقل إلى جزء السماح بالدخول.
TEST AL, 04H يفحص PA2، وهو إشارة البطاقة غير الصحيحة.
JNZ ACCESS_DENIED إذا كانت البطاقة غير صحيحة، ينتقل إلى جزء رفض الدخول.
JMP MAIN_LOOP إذا لا توجد إشارة بطاقة واضحة، يرجع البرنامج إلى المراقبة.
ACCESS_GRANTED: بداية جزء السماح بالدخول.
MOV DX, PORTB يحدد Port B لأنه يتحكم بموتور البوابة واللمبات.
MOV AL, 03H يضع 03H في AL، أي تشغيل PB0 و PB1. هذا يعني تشغيل الموتور واللمبة الخضراء.
OUT DX, AL يرسل 03H إلى Port B، فتفتح البوابة وتضيء اللمبة الخضراء.
MOV DX, PORTC يحدد Port C لعرض حالة البوابة.
MOV AL, 01H يضع 01H في AL، أي تشغيل PC0 الذي يمثل Gate Open.
OUT DX, AL يرسل 01H إلى Port C، فيظهر أن البوابة مفتوحة.
CALL GATE_DELAY يستدعي دالة تأخير حتى تبقى البوابة مفتوحة لفترة تسمح بالمرور.
MOV DX, PORTB بعد انتهاء التأخير، يحدد Port B لإغلاق البوابة.
MOV AL, 00H يضع صفر في AL لإطفاء الموتور واللمبة الخضراء.
OUT DX, AL يرسل 00H إلى Port B، فتتوقف البوابة عن الفتح.
MOV DX, PORTC يحدد Port C لتغيير حالة البوابة.
MOV AL, 02H يضع 02H في AL، أي إظهار أن البوابة مقفلة.
OUT DX, AL يرسل 02H إلى Port C، فتظهر حالة Gate Locked.
JMP MAIN_LOOP يرجع البرنامج إلى المراقبة بعد انتهاء عملية الدخول.
ACCESS_DENIED: بداية جزء رفض الدخول.
MOV DX, PORTB يحدد Port B لتشغيل لمبة الرفض والجرس.
MOV AL, 0CH يضع 0CH في AL، أي تشغيل PB2 و PB3. هذا يعني لمبة حمراء وجرس.
OUT DX, AL يرسل 0CH إلى Port B، فيتم تشغيل الرفض والجرس.
CALL SHORT_DELAY يعمل تأخيراً قصيراً والجرس واللمبة الحمراء شغالين.
MOV DX, PORTB يحدد Port B مرة أخرى.
MOV AL, 04H يضع 04H في AL، أي تشغيل اللمبة الحمراء فقط وإطفاء الجرس.
OUT DX, AL يرسل 04H إلى Port B، فتبقى لمبة الرفض مضاءة فقط.
CALL SHORT_DELAY يعمل تأخيراً قصيراً قبل إرجاع النظام للوضع الطبيعي.
MOV DX, PORTB يحدد Port B لإطفاء مخارج الرفض.
MOV AL, 00H يضع صفر في AL لإطفاء اللمبة والجرس.
OUT DX, AL يرسل 00H إلى Port B، فتعود المخارج للوضع الطبيعي.
MOV DX, PORTC يحدد Port C لتأكيد حالة القفل.
MOV AL, 02H يضع 02H في AL لإظهار أن البوابة مقفلة.
OUT DX, AL يرسل 02H إلى Port C، أي Gate Locked.
JMP MAIN_LOOP يرجع إلى المراقبة الرئيسية.
EMERGENCY_STOP: بداية جزء حالة الطوارئ.
MOV DX, PORTB يحدد Port B لأنه يتحكم بالموتور والجرس.
MOV AL, 0CH يضع 0CH في AL، أي تشغيل اللمبة الحمراء والجرس مع إيقاف الموتور.
OUT DX, AL يرسل 0CH إلى Port B، فتظهر حالة الخطر.
MOV DX, PORTC يحدد Port C لعرض مؤشر الطوارئ.
MOV AL, 04H يضع 04H في AL، أي تشغيل PC2 كمؤشر Emergency.
OUT DX, AL يرسل 04H إلى Port C، فيظهر أن النظام في حالة طوارئ.
EMERGENCY_LOOP: بداية حلقة البقاء في حالة الطوارئ.
CALL SHORT_DELAY يعمل تأخيراً قصيراً داخل حلقة الطوارئ.
MOV DX, PORTA يحدد Port A لقراءة زر الطوارئ مرة أخرى.
IN AL, DX يقرأ حالة Port A لمعرفة هل ما زال زر الطوارئ مضغوطاً.
TEST AL, 08H يفحص PA3 الخاص بزر Emergency Stop.
JNZ EMERGENCY_LOOP إذا كان زر الطوارئ لا يزال مضغوطاً، يبقى البرنامج داخل حلقة الطوارئ.
MOV DX, PORTB بعد انتهاء الطوارئ، يحدد Port B لإطفاء التحذيرات.
MOV AL, 00H يضع صفر في AL لإطفاء كل مخارج Port B.
OUT DX, AL يرسل 00H إلى Port B.
MOV DX, PORTC يحدد Port C لإرجاع حالة البوابة إلى القفل.
MOV AL, 02H يضع 02H في AL، أي Gate Locked.
OUT DX, AL يرسل 02H إلى Port C.
JMP MAIN_LOOP يرجع إلى حلقة المراقبة الرئيسية.
GATE_DELAY PROC بداية دالة التأخير التي تبقي البوابة مفتوحة لفترة.
PUSH CX يحفظ قيمة CX في Stack قبل استخدامه في التأخير.
MOV CX, 30 يجعل CX يساوي 30، أي سيتم تنفيذ التأخير القصير 30 مرة.
GD_LOOP: بداية حلقة تأخير فتح البوابة.
CALL SHORT_DELAY يستدعي دالة التأخير القصير.
LOOP GD_LOOP ينقص CX ويكرر الحلقة حتى تنتهي مدة فتح البوابة.
POP CX يعيد قيمة CX القديمة من Stack.
RET يرجع من دالة GATE_DELAY إلى مكان الاستدعاء.
GATE_DELAY ENDP نهاية دالة تأخير البوابة.
SHORT_DELAY PROC بداية دالة التأخير القصير.
PUSH CX يحفظ CX قبل استخدامه.
MOV CX, 0FFFFH يضع قيمة كبيرة في CX لصنع تأخير بسيط.
D1: بداية حلقة التأخير القصير.
LOOP D1 ينقص CX ويكرر الحلقة حتى يصبح CX صفراً.
POP CX يعيد قيمة CX السابقة.
RET يرجع من دالة التأخير القصير.
SHORT_DELAY ENDP نهاية دالة التأخير القصير.
END START نهاية البرنامج، والتنفيذ يبدأ من START.

ملخص سريع للمناقشة

في هذا المشروع تم استخدام 8255 كواجهة لنظام بوابة أمنية. Port A يعمل كمدخل لقراءة حساس وجود الشخص، إشارة البطاقة الصحيحة، إشارة البطاقة الخاطئة، وزر الطوارئ. Port B يتحكم في موتور البوابة واللمبات والجرس، بينما Port C يعرض حالة البوابة. البرنامج يعطي الطوارئ أعلى أولوية، ثم يتحقق من وجود الشخص، وبعدها من صحة البطاقة قبل فتح البوابة.
