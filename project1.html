Project 1

Power Surge & Fire Emergency System

نظام حماية من قفزات الجهد والحريق باستخدام 8086 و 8255

← الرجوع للصفحة الرئيسية

1. شرح المشكلة وفهم فكرتها

عند عودة التيار الكهربائي الحكومي بعد الانقطاع، لا يكون الجهد مستقراً مباشرة في أول لحظات الرجوع. أحياناً تحدث قفزة جهد مفاجئة تسمى Voltage Surge، وهذه القفزة قد تسبب تلف الأجهزة أو شرارة كهربائية أو ماس كهربائي داخل لوحة التوزيع.

بالإضافة إلى ذلك، إذا حدث دخان أو لهب داخل غرفة الكهرباء أو غرفة المنظومة، يجب أن يقوم النظام بفصل الكهرباء فوراً وتشغيل إنذار لتنبيه الموجودين.

فكرة المشروع ببساطة

نستخدم 8255 كوسيط بين المعالج 8086 وبين الحساسات والريليات. المعالج يقرأ حالة الحساسات من Port A، ويتحكم بالريلي الرئيسي عن طريق Port B، ويتحكم بالإنذار عن طريق Port C.

2. شرح الحل باستخدام 8255

يتم برمجة Port A كمدخل Input لأنه يستقبل إشارات من الحساسات. أما Port B و Port C فيتم برمجتهما كمخارج Output لأنهما يرسلان أوامر إلى الريلي والإنذار.

البرنامج يفحص حساس الحريق أولاً، لأن الحريق حالة طوارئ ولها أولوية أعلى من عودة الكهرباء. إذا اكتشف البرنامج وجود حريق، يرسل 00H إلى Port B لفصل الكهرباء، ثم يجعل Port C يشتغل ويطفي بشكل متكرر لتشغيل الإنذار.

أما إذا رجعت الكهرباء بدون وجود حريق، فالبرنامج لا يوصل الكهرباء مباشرة. يقوم أولاً بعمل Delay حتى يستقر الجهد، وبعد ذلك يفحص الحساسات مرة ثانية، فإذا كانت الأمور آمنة، يرسل 01H إلى Port B لتشغيل الريلي الرئيسي.

3. Interface Map

Port A = Input PA0 = Grid Return Sensor
PA1 = Smoke / Fire Sensor
Port B = Output PB0 = Main Relay
1 = Connect Power
0 = Cut Power
Port C = Output PC0 = Buzzer
1 = Alarm ON
0 = Alarm OFF

Control Word المستخدم هو 90H. معناه أن Port A مدخل Input، و Port B و Port C مخارج Output، وكلها تعمل في Mode 0.

4. الكود الكامل للمشكلة

ORG 100H

; -------------------------------
; 8255 Port Addresses
; -------------------------------
PORTA   EQU 300H
PORTB   EQU 301H
PORTC   EQU 302H
CTRL    EQU 303H

; -------------------------------
; Main Program
; -------------------------------
START:
    MOV DX, CTRL
    MOV AL, 90H
    OUT DX, AL

    ; Initially turn off all outputs
    MOV DX, PORTB
    MOV AL, 00H
    OUT DX, AL

    MOV DX, PORTC
    MOV AL, 00H
    OUT DX, AL

MAIN_LOOP:
    ; Read sensors from Port A
    MOV DX, PORTA
    IN  AL, DX

    ; Check fire sensor first
    TEST AL, 02H
    JNZ FIRE_EMERGENCY

    ; Check grid return sensor
    TEST AL, 01H
    JNZ GRID_RETURN

    JMP MAIN_LOOP

; -------------------------------
; Grid returned safely
; -------------------------------
GRID_RETURN:
    ; Keep house isolated during surge time
    MOV DX, PORTB
    MOV AL, 00H
    OUT DX, AL

    CALL DELAY_10_SEC

    ; After delay, check sensors again
    MOV DX, PORTA
    IN  AL, DX

    ; If fire appears, go to emergency
    TEST AL, 02H
    JNZ FIRE_EMERGENCY

    ; If grid is no longer present, return to monitoring
    TEST AL, 01H
    JZ MAIN_LOOP

    ; Grid is stable, connect the house
    MOV DX, PORTB
    MOV AL, 01H
    OUT DX, AL

    JMP MAIN_LOOP

; -------------------------------
; Fire Emergency State
; -------------------------------
FIRE_EMERGENCY:
    ; Cut off all power relays
    MOV DX, PORTB
    MOV AL, 00H
    OUT DX, AL

ALARM_LOOP:
    ; Turn buzzer ON
    MOV DX, PORTC
    MOV AL, 01H
    OUT DX, AL

    CALL SHORT_DELAY

    ; Turn buzzer OFF
    MOV DX, PORTC
    MOV AL, 00H
    OUT DX, AL

    CALL SHORT_DELAY

    JMP ALARM_LOOP

; -------------------------------
; Long delay, represents surge waiting time
; -------------------------------
DELAY_10_SEC PROC
    PUSH CX

    MOV CX, 50

D10_LOOP:
    CALL SHORT_DELAY
    LOOP D10_LOOP

    POP CX
    RET
DELAY_10_SEC ENDP

; -------------------------------
; Short delay for buzzer blinking
; -------------------------------
SHORT_DELAY PROC
    PUSH CX

    MOV CX, 0FFFFH

D1:
    LOOP D1

    POP CX
    RET
SHORT_DELAY ENDP

END START

5. شرح الكود سطر بسطر

السطر الشرح
ORG 100H يحدد أن البرنامج من نوع COM ويبدأ التنفيذ من العنوان 100H.
PORTA EQU 300H يعطي اسم PORTA للعنوان 300H، وهو عنوان Port A في 8255.
PORTB EQU 301H يعطي اسم PORTB للعنوان 301H، وهو عنوان Port B.
PORTC EQU 302H يعطي اسم PORTC للعنوان 302H، وهو عنوان Port C.
CTRL EQU 303H يعطي اسم CTRL للعنوان 303H، وهو Control Register الخاص بإعداد 8255.
START: بداية البرنامج الرئيسية.
MOV DX, CTRL يضع عنوان Control Register في المسجل DX حتى نرسل له Control Word.
MOV AL, 90H يضع قيمة Control Word في AL. القيمة 90H تجعل Port A مدخل، و Port B و Port C مخارج.
OUT DX, AL يرسل القيمة 90H إلى Control Register، وبذلك يتم ضبط اتجاه المنافذ.
MOV DX, PORTB يضع عنوان Port B داخل DX للتحكم بمخارج الريلي.
MOV AL, 00H يضع صفر في AL، وهذا يعني إطفاء جميع مخارج Port B.
OUT DX, AL يرسل 00H إلى Port B، أي فصل الريلي في بداية التشغيل.
MOV DX, PORTC يضع عنوان Port C داخل DX للتحكم بالإنذار.
MOV AL, 00H يضع صفر في AL من أجل إطفاء الإنذار.
OUT DX, AL يرسل 00H إلى Port C، أي أن الإنذار مطفأ في البداية.
MAIN_LOOP: بداية حلقة المراقبة المستمرة للحساسات.
MOV DX, PORTA يضع عنوان Port A في DX حتى يتم قراءة الحساسات منه.
IN AL, DX يقرأ قيمة Port A ويضعها في AL. هذه القيمة تمثل حالة الحساسات.
TEST AL, 02H يفحص البت رقم 1 في AL. هذا البت يمثل حساس الحريق PA1.
JNZ FIRE_EMERGENCY إذا كان حساس الحريق يعمل، يقفز البرنامج مباشرة إلى حالة الطوارئ.
TEST AL, 01H يفحص البت رقم 0 في AL. هذا البت يمثل حساس عودة الكهرباء PA0.
JNZ GRID_RETURN إذا كانت الكهرباء قد رجعت، ينتقل البرنامج إلى جزء معالجة عودة الكهرباء.
JMP MAIN_LOOP إذا لا يوجد حريق ولا رجعت الكهرباء، يرجع البرنامج للمراقبة مرة أخرى.
GRID_RETURN: هذا الجزء ينفذ عندما يكتشف النظام أن الكهرباء رجعت.
MOV DX, PORTB يحدد Port B لأنه المسؤول عن الريلي الرئيسي.
MOV AL, 00H يضع صفر في AL من أجل إبقاء الكهرباء مفصولة أثناء وقت الانتظار.
OUT DX, AL يرسل 00H إلى Port B، أي يمنع توصيل الكهرباء مباشرة عند رجوعها.
CALL DELAY_10_SEC يستدعي دالة تأخير لتمثيل فترة انتظار حتى يستقر الجهد.
MOV DX, PORTA بعد التأخير، يرجع يحدد Port A لقراءة الحساسات مرة ثانية.
IN AL, DX يقرأ حالة الحساسات مرة أخرى بعد الانتظار.
TEST AL, 02H يفحص هل ظهر حريق أثناء وقت الانتظار.
JNZ FIRE_EMERGENCY إذا ظهر حريق، يذهب فوراً لحالة الطوارئ.
TEST AL, 01H يفحص هل الكهرباء ما زالت موجودة بعد التأخير.
JZ MAIN_LOOP إذا لم تعد الكهرباء موجودة، يرجع البرنامج للمراقبة.
MOV DX, PORTB يحدد Port B من جديد من أجل تشغيل الريلي.
MOV AL, 01H يضع القيمة 01H في AL، وهي تعني تشغيل PB0 فقط.
OUT DX, AL يرسل 01H إلى Port B، أي تشغيل الريلي الرئيسي وتوصيل الكهرباء.
JMP MAIN_LOOP بعد توصيل الكهرباء يرجع البرنامج للمراقبة المستمرة.
FIRE_EMERGENCY: بداية جزء الطوارئ، وينفذ عند وجود حريق أو دخان.
MOV DX, PORTB يحدد Port B لأنه يتحكم بالريليات.
MOV AL, 00H يضع صفر في AL لفصل جميع المخارج.
OUT DX, AL يرسل 00H إلى Port B، أي فصل الكهرباء بالكامل.
ALARM_LOOP: بداية حلقة الإنذار المتقطع.
MOV DX, PORTC يحدد Port C لأنه متصل بالـ Buzzer.
MOV AL, 01H يضع 01H في AL لتشغيل PC0.
OUT DX, AL يرسل 01H إلى Port C، فيعمل الإنذار.
CALL SHORT_DELAY يعمل تأخير قصير والإنذار شغال.
MOV DX, PORTC يحدد Port C مرة أخرى.
MOV AL, 00H يضع صفر في AL لإطفاء الإنذار.
OUT DX, AL يرسل 00H إلى Port C، فيطفئ الإنذار.
CALL SHORT_DELAY يعمل تأخير قصير والإنذار مطفأ.
JMP ALARM_LOOP يرجع لبداية حلقة الإنذار، فيصبح الإنذار متقطعاً ON/OFF.
DELAY_10_SEC PROC بداية دالة التأخير الطويل.
PUSH CX يحفظ قيمة CX في Stack حتى لا تضيع قيمته الأصلية.
MOV CX, 50 يجعل CX يساوي 50، أي سيتم تكرار التأخير القصير 50 مرة.
D10_LOOP: بداية حلقة التأخير الطويل.
CALL SHORT_DELAY يستدعي التأخير القصير.
LOOP D10_LOOP ينقص CX بمقدار 1، وإذا لم يصل إلى صفر يرجع إلى D10_LOOP.
POP CX يرجع قيمة CX القديمة من Stack.
RET يرجع إلى المكان الذي استدعى الدالة.
DELAY_10_SEC ENDP نهاية دالة التأخير الطويل.
SHORT_DELAY PROC بداية دالة التأخير القصير.
PUSH CX يحفظ CX قبل استخدامه داخل دالة التأخير.
MOV CX, 0FFFFH يضع قيمة كبيرة في CX حتى تستغرق الحلقة وقتاً ملحوظاً.
D1: بداية حلقة التأخير القصير.
LOOP D1 ينقص CX ويكرر الحلقة إلى أن يصبح CX صفراً.
POP CX يعيد قيمة CX السابقة.
RET يرجع من دالة التأخير القصير.
SHORT_DELAY ENDP نهاية دالة التأخير القصير.
END START يخبر المترجم أن البرنامج انتهى وأن التنفيذ يبدأ من START.

ملخص سريع للمناقشة

في هذا المشروع تم استخدام 8255 كواجهة بين 8086 والحساسات الخارجية. Port A تم ضبطه كمدخل لقراءة عودة الكهرباء وحساس الحريق. Port B تم ضبطه كمخرج للتحكم في ريلي الكهرباء، وPort C كمخرج لتشغيل الإنذار. البرنامج يعطي أولوية للحريق، فإذا اشتغل حساس الحريق يتم فصل الكهرباء فوراً وتشغيل إنذار متقطع. أما عند عودة الكهرباء، فالبرنامج ينتظر فترة قبل توصيل الريلي لحماية الأجهزة من قفزة الجهد.
