Project 2

Voltage Sags & Sensitive Load Protection

حماية الأجهزة الحساسة من هبوط وتذبذب الجهد باستخدام 8086 و 8255

← الرجوع للصفحة الرئيسية

1. شرح المشكلة وفهم فكرتها

في المنازل التي تعتمد على العاكس أو الطاقة البديلة، قد يحدث هبوط مفاجئ في الجهد عند تشغيل أحمال قوية مثل الثلاجة أو المضخة. هذا الهبوط يسمى Voltage Sag، وقد يكون مؤقتاً لكنه يؤثر على الأجهزة الحساسة.

الأجهزة مثل الشاشة الذكية، الرسيفر، الراوتر، ولوحات التحكم تحتوي على دوائر إلكترونية حساسة. إذا اشتغلت هذه الأجهزة أثناء انخفاض الجهد أو تذبذبه، قد تتلف المكثفات أو الدوائر المتكاملة داخلها.

فكرة المشروع ببساطة

النظام يراقب حالة الجهد من خلال Port A. إذا كان الجهد منخفضاً، يقوم بفصل الأجهزة الحساسة فقط ويبقي الإضاءة الأساسية تعمل. وإذا عاد الجهد إلى الوضع الآمن واستقر لعدة قراءات متتالية، يرجع تشغيل الأجهزة الحساسة.

2. شرح الحل باستخدام 8255

نستخدم دائرة Comparator أو ADC لتحويل حالة الجهد إلى إشارات رقمية آمنة تدخل إلى Port A في 8255. المعالج 8086 يقرأ هذه الإشارات باستمرار باستخدام أمر IN.

إذا كانت إشارة الجهد المنخفض مفعلة، يرسل المعالج قيمة معينة إلى Port B تفصل الأحمال الحساسة فقط. وفي نفس الوقت يرسل حالة إلى Port C لتشغيل لمبة تحذير أو مؤشر أن الأحمال الحساسة مفصولة.

إذا كانت إشارة الجهد المستقر مفعلة، لا يعيد النظام تشغيل الأجهزة مباشرة، بل يتأكد من الاستقرار عدة مرات باستخدام Loop و Delay. هذا يمنع تشغيل الأجهزة إذا كان الجهد ما زال يتذبذب.

3. Interface Map

Port A = Input PA0 = Voltage Low Signal
PA1 = Voltage Stable Signal
Port B = Output PB0 = Basic Lights Relay
PB1 = TV Relay
PB2 = Receiver Relay
PB3 = Router Relay
Port C = Output PC0 = Voltage Low LED
PC1 = Voltage Stable LED
PC2 = Sensitive Loads OFF LED

Control Word المستخدم هو 90H. معناه أن Port A مدخل Input، و Port B و Port C مخارج Output، وكلها تعمل في Mode 0.

4. الكود الكامل للمشكلة

ORG 100H

; -------------------------------
; 8255 Port Addresses
; -------------------------------
PORTA   EQU 300H
PORTB   EQU 301H
PORTC   EQU 302H
CTRL    EQU 303H

; -------------------------------
; Main Program
; -------------------------------
START:
    MOV DX, CTRL
    MOV AL, 90H
    OUT DX, AL

    ; Initially turn on all loads
    MOV DX, PORTB
    MOV AL, 0FH
    OUT DX, AL

    ; Show stable status at beginning
    MOV DX, PORTC
    MOV AL, 02H
    OUT DX, AL

MAIN_LOOP:
    ; Read voltage status from Port A
    MOV DX, PORTA
    IN  AL, DX

    ; Check if voltage is low
    TEST AL, 01H
    JNZ VOLTAGE_LOW

    ; Check if voltage is stable
    TEST AL, 02H
    JNZ VOLTAGE_STABLE

    JMP MAIN_LOOP

; -------------------------------
; Low voltage state
; -------------------------------
VOLTAGE_LOW:
    ; Keep lights ON only, disconnect sensitive loads
    MOV DX, PORTB
    MOV AL, 01H
    OUT DX, AL

    ; PC0 = Low voltage LED, PC2 = Loads disconnected LED
    MOV DX, PORTC
    MOV AL, 05H
    OUT DX, AL

    CALL SHORT_DELAY
    JMP MAIN_LOOP

; -------------------------------
; Stable voltage state
; -------------------------------
VOLTAGE_STABLE:
    ; Require 5 consecutive stable readings
    MOV CX, 05H

STABLE_CHECK:
    MOV DX, PORTA
    IN  AL, DX

    ; If voltage becomes low again, protect loads immediately
    TEST AL, 01H
    JNZ VOLTAGE_LOW

    ; If stable signal disappears, go back to monitoring
    TEST AL, 02H
    JZ MAIN_LOOP

    CALL SHORT_DELAY
    LOOP STABLE_CHECK

    ; Restore all loads after confirmed stability
    MOV DX, PORTB
    MOV AL, 0FH
    OUT DX, AL

    ; PC1 = Stable LED
    MOV DX, PORTC
    MOV AL, 02H
    OUT DX, AL

    JMP MAIN_LOOP

; -------------------------------
; Short delay
; -------------------------------
SHORT_DELAY PROC
    PUSH CX

    MOV CX, 0FFFFH

D1:
    LOOP D1

    POP CX
    RET
SHORT_DELAY ENDP

END START

5. شرح الكود سطر بسطر

السطر الشرح
ORG 100H يحدد أن البرنامج من نوع COM ويبدأ من العنوان 100H.
PORTA EQU 300H يعطي اسم PORTA للعنوان 300H، وهو المنفذ الذي سيقرأ حالة الجهد.
PORTB EQU 301H يعطي اسم PORTB للعنوان 301H، وهو المنفذ الذي يتحكم بريليات الأحمال.
PORTC EQU 302H يعطي اسم PORTC للعنوان 302H، وهو المنفذ الخاص بمؤشرات الحالة.
CTRL EQU 303H يعطي اسم CTRL للعنوان 303H، وهو Control Register المستخدم لضبط 8255.
START: بداية تنفيذ البرنامج.
MOV DX, CTRL يضع عنوان Control Register داخل DX.
MOV AL, 90H يضع Control Word في AL. القيمة 90H تجعل Port A مدخل و Port B و Port C مخارج.
OUT DX, AL يرسل Control Word إلى 8255 حتى يتم ضبط اتجاه المنافذ.
MOV DX, PORTB يضع عنوان Port B في DX لأننا سنرسل له أوامر تشغيل الأحمال.
MOV AL, 0FH يضع القيمة 0FH في AL. هذه القيمة تعني تشغيل أول أربع بتات PB0 إلى PB3.
OUT DX, AL يرسل 0FH إلى Port B، أي تشغيل الإضاءة والأجهزة الحساسة في البداية.
MOV DX, PORTC يضع عنوان Port C في DX للتحكم بمؤشرات الحالة.
MOV AL, 02H يضع 02H في AL، أي تشغيل PC1 كمؤشر أن الجهد مستقر.
OUT DX, AL يرسل 02H إلى Port C لإظهار حالة الاستقرار في البداية.
MAIN_LOOP: بداية حلقة المراقبة المستمرة لحالة الجهد.
MOV DX, PORTA يضع عنوان Port A في DX حتى يقرأ إشارات الجهد.
IN AL, DX يقرأ حالة Port A ويضعها في AL.
TEST AL, 01H يفحص البت PA0. إذا كان يساوي 1 فهذا يعني أن الجهد منخفض.
JNZ VOLTAGE_LOW إذا كان الجهد منخفضاً، ينتقل البرنامج إلى جزء الحماية VOLTAGE_LOW.
TEST AL, 02H يفحص البت PA1. إذا كان يساوي 1 فهذا يعني أن الجهد مستقر.
JNZ VOLTAGE_STABLE إذا كان الجهد مستقراً، ينتقل البرنامج إلى جزء التأكد من الاستقرار.
JMP MAIN_LOOP إذا لم توجد إشارة واضحة، يرجع البرنامج للمراقبة من جديد.
VOLTAGE_LOW: بداية الجزء الخاص بحالة انخفاض الجهد.
MOV DX, PORTB يحدد Port B لأنه يتحكم بريليات الأحمال.
MOV AL, 01H يضع 01H في AL، أي تشغيل PB0 فقط وإطفاء PB1 و PB2 و PB3.
OUT DX, AL يرسل 01H إلى Port B، فيبقي الإضاءة فقط ويفصل الأجهزة الحساسة.
MOV DX, PORTC يحدد Port C لإظهار مؤشرات التحذير.
MOV AL, 05H يضع 05H في AL، أي تشغيل PC0 و PC2. معناها جهد منخفض والأحمال الحساسة مفصولة.
OUT DX, AL يرسل 05H إلى Port C لتشغيل لمبات التحذير المناسبة.
CALL SHORT_DELAY ينفذ تأخيراً قصيراً قبل الرجوع للمراقبة.
JMP MAIN_LOOP يرجع إلى حلقة المراقبة الرئيسية.
VOLTAGE_STABLE: بداية الجزء الخاص بحالة استقرار الجهد.
MOV CX, 05H يضع 5 في CX، أي يجب التأكد من الاستقرار خمس مرات متتالية.
STABLE_CHECK: بداية حلقة فحص الاستقرار المتكرر.
MOV DX, PORTA يحدد Port A حتى يقرأ حالة الجهد مرة أخرى.
IN AL, DX يقرأ إشارة الجهد من Port A إلى AL.
TEST AL, 01H يفحص هل عاد الجهد للانخفاض أثناء مرحلة التأكد.
JNZ VOLTAGE_LOW إذا أصبح الجهد منخفضاً، ينتقل مباشرة إلى الحماية ويفصل الأجهزة الحساسة.
TEST AL, 02H يفحص هل إشارة الاستقرار ما زالت موجودة.
JZ MAIN_LOOP إذا اختفت إشارة الاستقرار، يرجع إلى المراقبة ولا يشغل الأجهزة الحساسة.
CALL SHORT_DELAY يعمل تأخيراً قصيراً بين كل قراءة والثانية.
LOOP STABLE_CHECK ينقص CX ويكرر الفحص حتى تكتمل خمس قراءات مستقرة.
MOV DX, PORTB بعد التأكد من الاستقرار، يحدد Port B لإعادة تشغيل الأحمال.
MOV AL, 0FH يضع 0FH في AL لتشغيل PB0 و PB1 و PB2 و PB3.
OUT DX, AL يرسل 0FH إلى Port B، وبذلك تعود الأجهزة الحساسة للعمل.
MOV DX, PORTC يحدد Port C لتحديث مؤشرات الحالة.
MOV AL, 02H يضع 02H في AL لتشغيل مؤشر الجهد المستقر فقط.
OUT DX, AL يرسل 02H إلى Port C، أي أن النظام في حالة آمنة ومستقرة.
JMP MAIN_LOOP يرجع البرنامج للمراقبة المستمرة بعد إعادة تشغيل الأحمال.
SHORT_DELAY PROC بداية دالة التأخير القصير.
PUSH CX يحفظ قيمة CX في Stack حتى لا تضيع أثناء التأخير.
MOV CX, 0FFFFH يضع قيمة كبيرة في CX حتى تستغرق الحلقة وقتاً بسيطاً.
D1: بداية حلقة التأخير.
LOOP D1 ينقص CX ويكرر الحلقة حتى يصبح CX صفراً.
POP CX يعيد قيمة CX القديمة بعد انتهاء التأخير.
RET يرجع من دالة التأخير إلى مكان الاستدعاء.
SHORT_DELAY ENDP نهاية دالة التأخير القصير.
END START نهاية البرنامج، والتنفيذ يبدأ من START.

ملخص سريع للمناقشة

في هذا المشروع استخدمنا 8255 لمراقبة حالة الجهد وحماية الأجهزة الحساسة. Port A يعمل كمدخل لقراءة إشارات انخفاض واستقرار الجهد. Port B يعمل كمخرج للتحكم في ريليات الأجهزة، وPort C يعمل كمخرج لمؤشرات الحالة. إذا كان الجهد منخفضاً، يفصل البرنامج الأجهزة الحساسة ويبقي الإضاءة فقط. وإذا أصبح الجهد مستقراً، لا يعيد التشغيل مباشرة، بل يتأكد من الاستقرار خمس مرات متتالية قبل إعادة تشغيل الأجهزة.
